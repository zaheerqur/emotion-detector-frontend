About this project

What it is

Built for COMP 472 (Artificial Intelligence) at Concordia University. The goal was to design, train, and evaluate a convolutional neural network for facial emotion recognition from scratch — no pretrained weights, no transfer learning. The model classifies faces into four categories: Angry, Engaged, Happy, and Neutral.

How it was trained

The CNN was trained on the FER2013 benchmark dataset — over 35,000 grayscale 48×48 facial images. The architecture uses three convolutional blocks with BatchNorm and LeakyReLU activations, followed by fully connected layers with dropout. Three variants were evaluated; this one performed best on validation.

How it's deployed

The trained PyTorch model was exported to ONNX format for serving — a standard production practice that cuts the runtime footprint from ~500MB to ~50MB without changing model behaviour. The backend runs on Flask via Render; this frontend is hosted on Netlify.

Why the samples

The model was trained and evaluated on FER2013, so it performs most reliably on images from that distribution. The sample grid pulls directly from the dataset used during training. You can still upload your own photo — just know that real-world images will naturally produce less consistent results.
